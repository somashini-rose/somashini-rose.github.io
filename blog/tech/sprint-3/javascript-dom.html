JavaScript and its relationship to HTML and CSS.

While researching JavaScript I came across a pretty cool analogy which is by far my favourite.
In this analogy we liken a website to a human body.
HTML is the structure or skelton, the bones if your will. It holds everything together and provides a base to hang everything off.
CSS is the skin, the outside appearance the thing other humans can touch and feel the tangible element
With just these two elements alone you can have a funtional pretty website, but if you want it to do more than just basic functionality you need JavaScript.
JavaScript ('JS') is the thing that pulls it all together, in our analogy JS is the muscles in the human body it provides motion and behaviour, it acts out the heavy lifting. JS takes your basic, pretty looking body up a notch and makes it interactive.

What is Control Flow and Loops?

Control flow in JavaScript is how your computer runs code from top to bottom.
It starts from the first line and ends at the last line, unless it hits any statement that changes the flow of the program such as loops, conditionals, or functions. An example of a control flow is using a recipe, you start reading and following a recipe from top to bottom in sequentuial order, sometimes that flow is interrupted by a choice or substitution of an ingrediant or method (conditional - if/else statement).
Once the condition has been met the flow continues in sequential order.

Loops are iteration statements that will keep running until there is either nothing left to loop over, or if the condition becomes false.

What is The DOM and how you can interact with it?

The Document Object Model ('DOM') is a tree-like representation (very hierachical) of the web page that gets loaded into the browser.

It is a web API through which JavaScript interacts with HTML. There are many properties and methods which you can use to get information about the DOM and manipulate it.
You use the API directly in JavaScript from within what is called a script, a program run by a browser. When you create a script, whether inline in a script element or by linking a web page.

Accessing data from arrays and objects.

Both objects and arrays are considered “special” in JavaScript. Objects and arrays can be used to store a collection of data (rather than just a single value) in a container.
Objects are used to represent a “thing” e.g. aperson, a car, a building, a book, a character in a game — basically anything that is made up or can be defined by a set of characteristics. In objects, these characteristics are called properties that consist of a key and a value.

What are Functions and why are they helpful

A function is a set of statements that performs a task or calculates a value, but for a procedure to qualify as a function, it should take some input and return an output where there is some obvious relationship between the input and the output. Functions allow you to define a block of code, give it a name and then execute it as many times as you want, which is useful in avoiding duplication of code making it easier to fix bugs etc.
