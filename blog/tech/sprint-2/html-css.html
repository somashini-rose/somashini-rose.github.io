What are the differences between relative, absolute, and fixed positioning?

The position property is used in CSS to define how an element is positioned on a webpage.

There are several values of positioning that can be used.

We will look at the three most commonly used positions; relative, absolute, and fixed.>

Relative: an element is positioned relative to its normal position.

You use this value to set the element to be positioned relative to other elements on the page.

Absolute: an element is positioned absolutely to it its first parent position or if that is not present it will relate directly to the HTML element.

This value allows you to position an element precisely where you want it.

Fixed: an element that is positioned in relation to the viewport(browser window).

The value is used to fix an element in place, it will remain in the same place even when the page is scrolled.

To position the element you must use top, right, bottom, left properties. The top or bottom property must be set, if this is not set the element will not exist on the page.

This is quite a handy position to use for things like a navigation bar.
